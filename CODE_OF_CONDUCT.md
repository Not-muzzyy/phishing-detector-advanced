# Code of Conduct

## Our Standards

Examples of behavior that contributes to a positive environment:

- being respectful
- accepting constructive feedback
- focusing on collaboration
- helping new contributors
- maintaining professionalism

Examples of unacceptable behavior:

- harassment
- discrimination
- trolling
- personal attacks
- abusive behavior
- disruptive conduct

## Enforcement

Project maintainers may remove comments, commits, issues, or pull requests that violate this code of conduct.

Repeated violations may result in temporary or permanent restrictions from the project.

## Scope

This code of conduct applies to:
- issues
- pull requests
- discussions
- project communication spaces

## Responsible Security Research

This repository supports educational and defensive cybersecurity research.

Contributors are expected to follow ethical and legal guidelines when participating in the project.
