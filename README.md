# Intelligent Phishing Detection System

A cybersecurity project that detects phishing using
machine learning without hardcoded rules.

## Features
- Text-based phishing detection
- Website URL feature analysis
- Meta-model decision system
- Fully data-driven learning

## Project Structure
- data/            → datasets
- models/          → trained ML models
- text_analysis/   → text model training
- website_analysis/→ website model training
- final_predict.py → unified prediction script

## How to Run
```bash
python final_predict.py
