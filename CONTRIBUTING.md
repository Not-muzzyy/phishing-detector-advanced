# Contributing

Thank you for contributing to Phishing Detector Advanced.

## Before contributing

Please:
- read the README
- review LEARN.md
- follow ethical usage guidelines
- avoid submitting malicious payloads

## Development setup

```bash
git clone https://github.com/Not-muzzyy/phishing-detector-advanced
cd phishing-detector-advanced
pip install -r requirements.txt
streamlit run phishing_dashboard.py
```

## Contribution workflow

1. Fork the repository
2. Create a new branch
3. Make changes
4. Test locally
5. Commit changes
6. Push your branch
7. Open a pull request

## Good contribution areas

- phishing detection improvements
- multilingual support
- UI/UX enhancements
- model optimization
- performance improvements
- documentation
- test coverage
- accessibility

## Pull request guidelines

Please:
- keep pull requests focused
- write clear commit messages
- explain why changes are needed
- include screenshots for UI changes
- avoid unrelated modifications

## Reporting issues

When reporting bugs, include:
- reproduction steps
- expected behavior
- screenshots or logs
- environment details

## Security notice

This project is for:
- cybersecurity education
- phishing awareness
- defensive research

Do not contribute:
- phishing kits
- credential stealers
- offensive payloads
- malicious automation
