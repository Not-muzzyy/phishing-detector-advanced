# Security Policy

## Supported Versions

Only the latest version of this repository is actively supported.

## Reporting Vulnerabilities

If you discover a security issue:

- do not publish it publicly
- avoid posting exploit details in issues
- contact the maintainer directly
- provide reproduction details if possible

Please include:
- affected component
- reproduction steps
- expected behavior
- screenshots or logs if available

## Responsible Disclosure

Security reports will be reviewed and addressed as quickly as possible.

Please allow reasonable time before public disclosure.

## Ethical Usage

This repository is intended for:
- cybersecurity education
- phishing awareness
- defensive security research
- safe testing environments

This repository must not be used for:
- phishing campaigns
- credential theft
- malicious automation
- bypassing security systems
- unauthorized access attempts

## Safe Testing Recommendations

Use:
- local environments
- disposable test accounts
- intentionally created phishing samples
- sandboxed systems

Avoid:
- production systems
- real user accounts
- unauthorized infrastructure
